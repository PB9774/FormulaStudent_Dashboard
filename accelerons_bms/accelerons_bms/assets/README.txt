ACCELERONS ELECTRIC — Logo Assets
──────────────────────────────────
Place your logo file in this directory.

Supported filenames (in order of priority):
  logo.png   ← recommended (transparent background preferred)
  logo.jpg
  logo.jpeg
  logo.bmp

The splash screen will automatically pick it up on next launch.
Recommended size: 600×120 px or wider (will be scaled automatically).

If no file is found, the built-in painted logo is used as fallback.
